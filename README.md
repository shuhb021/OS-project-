# SkillSphere Journal Portal

The official academic publishing portal for **SkillSphere**, presented by **Symbiosis University of Applied Sciences**.

## Features
- **University Branding**: Professional academic layout with Symbiosis Maroon and Crimson Red theme.
- **Journal Repository**: Explore high-quality research across multiple applied science disciplines.
- **Author Submission Portal**: Multi-step manuscript submission process for scholars.
- **Interactive Metrics**: Impact factor trends and global citation statistics visualized with Recharts.
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop viewing.

## Running Locally

1. **Prerequisites**: Ensure [Node.js](https://nodejs.org/) is installed.
2. **Setup**:
   - Clone the repository.
   - Run `npm install` in your terminal to install dependencies.
3. **Execution**:
   - Start your development server (e.g., `npm run dev` or `npx vite`).
   - Open the local URL in your browser.

*Note: The project uses a flat directory structure where the root serves as the source folder. For production deployment, consider migrating source files to a `src/` directory.*
